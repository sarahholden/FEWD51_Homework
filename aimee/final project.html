FINAL PROJECT IDEA

    TITLE
      Multimedia book Multi:zine


    THE IDEA
      Use as various multimedia sources as possible that the artist has created hands-on,
      using her studio practice such as hand-written caligraphy, and hand-made drawing and painting.


    OBJECTIVE
      To create "multimedia art" to demonstrate the artist's COMBINED skillset as
      1) a front end web developer and 2) graphic/animation artist in line with
      the Post-internet art aesthetics


    MEDIA
      CODING
      - HTML & CSS of course to power and stylize the multimedia canvas on the Web
      Plus JavaScript to illustrate interactive objects with the creative but efficient user interface in mind

      SOUND
      - JavaScript oscillator
      - Edited sound file of various artists
      - Apply various types of sound controls to enable interactive and more engaging user interface

      STUDIO ART
      Multiple sources of studio art such as
      - Caligraphy
      - Drawing and painting


    INTERFACE
